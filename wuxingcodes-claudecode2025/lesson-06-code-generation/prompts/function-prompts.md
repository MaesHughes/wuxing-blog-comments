# 函数生成Prompt集合

## 基础函数模板

### 1. 简单函数
```bash
创建一个[功能描述]函数：
- 接收参数：[参数列表]
- 返回值：[返回类型]
- 包含错误处理
- 添加参数验证
- 使用[编程语言]

示例：
创建一个计算圆面积的函数：
- 接收参数：半径（number）
- 返回值：面积（number）
- 包含错误处理
- 添加参数验证
- 使用JavaScript
```

### 2. 异步函数
```bash
创建一个异步[功能描述]函数：
- 使用[fetch/axios/async/await]
- 包含完整的错误处理
- 支持取消请求
- 添加超时处理
- 返回Promise对象

示例：
创建一个异步获取用户数据的函数：
- 使用fetch API
- 包含完整的错误处理
- 支持取消请求
- 添加超时处理（5秒）
- 返回Promise对象
```

### 3. 工具函数
```bash
生成一个[功能]工具函数库：
包含以下函数：
1. [函数1功能]
2. [函数2功能]
3. [函数3功能]

要求：
- 使用TypeScript
- 提供完整的类型定义
- 添加JSDoc注释
- 包含单元测试
- 遵循函数式编程
```

## 高级函数模板

### 1. 高阶函数
```bash
创建一个高阶函数，实现[功能描述]：
- 接收函数作为参数
- 返回新的函数
- 支持函数组合
- 保留原函数上下文
- 添加缓存机制

示例：

- 接收函数作为参数
- 返回新的函数
- 支持函数组合
- 保留原函数上下文
- 添加LRU缓存机制
```

### 2. 递归函数
```bash
创建一个递归函数解决[问题]：
- 定义基本情况
- 处理递归情况
- 优化尾递归
- 添加最大深度限制
- 包含性能测试

示例：
创建一个递归函数解决阶乘计算：
- 定义基本情况（0! = 1）
- 处理递归情况（n! = n * (n-1)!）
- 优化尾递归
- 添加最大深度限制（100）
- 包含性能测试
```

### 3. 算法函数
```bash
实现[算法名称]算法：
- 时间复杂度：O(n)

- 性能优化
- 可视化过程

示例：
实现快速排序算法：
- 时间复杂度：O(n log n)
- 空间复杂度：O(log n)
- 边界情况处理
- 性能优化（小数组使用插入排序）
- 可视化过程
```
