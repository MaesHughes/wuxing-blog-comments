## 🚀 快速开始

### 环境要求

- Python 3.8+
- Node.js 16+
- Git
- VS Code（推荐）

### 安装步骤

1. 克隆课程仓库
   ```bash
   git clone https://github.com/MaesHughes/wuxing-blog-comments.git
   cd wuxingcodes-claudecode2025
   ```

2. 查看第1课课件
   ```bash
   cd lesson-01-installation
   ```

3. 运行安装脚本
   - Windows: `.\install-windows.ps1`
   - macOS/Linux: `./install-unix.sh`
