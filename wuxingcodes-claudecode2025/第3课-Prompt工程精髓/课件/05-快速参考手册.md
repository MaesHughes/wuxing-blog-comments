## 📱 常用参数说明

### 性能参数
```bash
响应时间：<100ms (API), <50ms (UI)
内存使用：<512MB (应用), <100MB (组件)
并发量：1000 QPS (API), 100 并发 (UI)
包大小：<1MB (JS), <500KB (CSS)
```

### 安全参数
```bash
密码强度：8位以上，包含大小写、数字、特殊字符
Token有效期：访问令牌2小时，刷新令牌7天
API限流：100次/分钟/用户
文件上传：最大10MB，限制文件类型
```

### 兼容性参数
```bash
浏览器：Chrome 70+, Firefox 65+, Safari 12+, Edge 79+
设备：桌面、平板、移动端
分辨率：1920x1080 ~ 375x667
网络：2G/3G/4G/WiFi
```
